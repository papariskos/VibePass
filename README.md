# VibePass
🎫 Neon Rhythm Ticketing Platform (Εφαρμογή Συναυλιών)
Καλώς ήρθατε στο repository του Neon Rhythm! Αυτή η εφαρμογή αποτελεί ένα ολοκληρωμένο, σύγχρονο σύστημα (super-app) για αναζήτηση, αγορά και διαχείριση εισιτηρίων συναυλιών, τόσο για Χρήστες όσο και για Διοργανωτές.

🚀 Βασικά Χαρακτηριστικά (Features)
🔎 Προηγμένη Αναζήτηση: Άμεση εύρεση καλλιτεχνών / μουσικών γεγονότων.
🎟️ Αγορά & Μεταβίβαση: Ασφαλής αγορά μέσω Payment System και δυνατότητα αυτόματης μεταπώλησης/εκχώρησης (Transfer) για αποφυγή απάτης.
⏳ Λίστα Αναμονής (Smart Waitlist): Αυτόματη απόδοση ακυρωμένων εισιτηρίων στους επόμενους τυχερούς πελάτες, χωρίς ανθρώπινη παρέμβαση!
💎 Premium Υπηρεσίες: VIP Upgrades της τελευταίας στιγμής.
⚙️ Διαχείριση για Διοργανωτές (Operations): Δημιουργία events, Push Notifications σε περίπτωση αλλαγών (αναβολές) και αυτόματες επιστροφές χρημάτων.
